## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```
- expo
  ```sh
  npm install -g expo-cli
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/christina003/food_mood.git
   ```
2. Install NPM packages
   ```sh
   npm install
   cd backend
   npm install
   ```
3. Start Application
   ```sh
   expo start
   ```
4. Start Server
   ```sh
   cd backend
   npm start
   ```
